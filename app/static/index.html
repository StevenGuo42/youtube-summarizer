Help

Browse

Paste a YouTube URL into the search bar to fetch video information. Supports single videos, channels, and playlists.

For channels, you can filter by visibility (public, members-only, or all). Select one or more videos, configure processing options, then click Add to Queue.

Queue

Shows all processing jobs and their current status. Active jobs display a progress bar showing which pipeline step is running.

You can cancel active jobs, rerun failed or cancelled jobs, select jobs for bulk deletion, or clear all finished jobs.

Summaries

View completed summaries in compact or list view. Click a summary to expand and see its TL;DR, full summary, and transcript. You can export summaries as markdown or delete them.

Settings

Cookies
Upload a cookies.txt file to access members-only content. You can drag and drop, browse for a file, or paste the content directly.
LLM Model
The Claude model used for summarization (e.g. claude-sonnet-4-20250514).
Output Language
Set a language for summaries (e.g. "Japanese", "Spanish"). Leave empty to match the video's language.
Custom Prompt
Insert mode adds your instructions to the default prompt. Replace mode uses your text as the entire prompt.
Claude Authentication
Shows whether Claude OAuth is configured. Run claude auth login in your terminal to authenticate.

Processing Options

Dedup Mode
Regular — Removes similar keyframes using perceptual hashing. Good for most videos.
Slides — Uses structural similarity (SSIM). Better for presentations and slideshows.
OCR — Compares extracted text from keyframes. Best when on-screen text changes indicate new content.
None — Keeps all extracted keyframes.
Keyframe Context Mode
Image — Sends keyframe images to the LLM for visual context.
OCR — Sends OCR-extracted text files instead of images.
OCR + Image — Sends both OCR text files and images.
OCR Inline — Embeds OCR text directly in the transcript.
OCR Inline + Image — Embeds OCR text in the transcript and sends images.
None — Uses transcript only, no keyframe data.
Output Language
Override the default output language for this job. Leave empty to use the default from Settings.
